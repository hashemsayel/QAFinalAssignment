# Playwright SauceDemo Testing Project

## Website
https://www.saucedemo.com/

## Covered features
- Login
- Add to cart with one item and multiple items
- Checkout with one item and multiple items
- Remove from cart with one item and multiple items
- Sort from A-Z and price high to low

## Used concepts
- Playwright with TypeScript
- Page Object Model
- Hooks
- Parameterized tests
- Environment variables
- Browser state
- Test grouping
- Multi-browser testing with Chromium and Firefox

## Installation
```bash
npm install
npx playwright install
```

## Run tests
```bash
npm test
```

## Show report
```bash
npm run report
```
